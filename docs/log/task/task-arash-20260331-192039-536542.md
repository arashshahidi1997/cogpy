## Prompt

You are implementing the travelling-wave detection and analysis layer for the `cogpy.wave` subpackage. This adds spatial propagation analysis (direction, speed, wavelength estimation) to an existing waveform morphology package.

### Rules

1. **Do not modify** existing files: `detect.py`, `features.py`, `process.py`, `plot.py`, `utils.py` — they are waveform morphology tools and must remain unchanged.
2. **Update** `__init__.py` to register new submodules via `lazy_loader.attach()` (follow the pattern in `cogpy/spectral/__init__.py` and `cogpy/measures/__init__.py`).
3. **Conventions**: xarray DataArray with dims `"time"`, `"AP"`, `"ML"` (grids) or `"time"`, `"ch"` (flat). Sampling rate via `sig.fs`. NumPy-style docstrings. No file I/O in compute functions.
4. **Optional deps**: scikit-image for optical flow — guard with `from cogpy.utils.imports import import_optional; skimage = import_optional("skimage", pip_name="scikit-image")`.
5. **Tests** under `tests/wave/` mirroring source. Use pytest. Every estimator must have a synthetic round-trip test.

### Referencing papers and code

**Biblio**: The project has a `travelling-waves` biblio collection. When a function implements a method from a specific paper, cite it in the docstring like:

```python
def pgd(phase, geometry):
    """Phase-gradient directionality score.

    Measures how consistently the spatial phase gradient points in a single
    direction, following the method of Zhang et al. (2018) [1]_.

    References
    ----------
    .. [1] Zhang et al., "Theta and Alpha Oscillations Are Traveling Waves
       in the Human Neocortex", Neuron, 2018. DOI: 10.1016/j.neuron.2018.05.019
    """
```

Key paper-to-function mappings:
- Phase gradient / plane-wave fit / PGD: zhang_2018 (DOI: 10.1016/j.neuron.2018.05.019)
- Generalized phase: davis_2020 (DOI: 10.1038/s41586-020-2802-y)
- Optical flow for neural waves: afrashteh_2017 (DOI: 10.1016/j.neuroimage.2017.03.034)
- Pattern classification / vector fields: townsend_2018 (DOI: 10.1371/journal.pcbi.1006643)
- Capon beamforming: capon_1969 (DOI: 10.1109/PROC.1969.7278)
- Multidimensional multitaper: hanssen_1997 (DOI: 10.1016/S0165-1684(97)00076-5)
- Multitaper foundations: thomson_1982 (DOI: 10.1109/PROC.1982.12433)
- Rotating waves in PFC: bhattacharya_2022 (DOI: 10.1371/journal.pcbi.1009827)

**Codio**: Reference MATLAB toolboxes are cloned locally. Read their source to understand algorithms before porting. Do NOT wrap or call MATLAB — reimplement in pure NumPy/SciPy.

- `/storage/share/codelib/sayak66--wm_travelingwaves_code` — phase gradient, PGD, plane-wave fit, circular stats. Port the core fitting logic.
- `/storage/share/codelib/mullerlab--generalized-phase` — generalized phase estimation. Port the analytic-signal centering and negative-freq correction.
- `/storage/share/codelib/braindynamicsusyd--neuropatttoolbox` — optical flow on phase maps, velocity-field div/curl/critical points, pattern classification, surrogate generation. Port the classification and vector-field analysis logic.
- `/storage/share/codelib/navvab-afrashteh--ofamm` — Horn-Schunck / CLG optical flow. Use as algorithm reference; for implementation prefer wrapping scikit-image solvers.
- `/storage/share/codelib/evolutionaryneuralcodinglab--travelling-waves-or-sequentially-activated-discrete-modules` — surrogate/null-model generation. Port phase-randomization and shuffle logic.

When porting, add a note in the docstring:

```python
def compute_flow(frames, method="tvl1"):
    """Estimate dense velocity field via optical flow.

    Adapted from NeuroPattToolbox [1]_ using scikit-image solvers.

    References
    ----------
    .. [1] Townsend & Gong, "Detection and analysis of spatiotemporal patterns
       in brain activity", PLOS Comp Biol, 2018. DOI: 10.1371/journal.pcbi.1006643
    """
```

### New files to create

**`_types.py`** — Shared data containers
- `Geometry` dataclass: `dx`/`dy` floats for regular grids, OR `coords` array (N,2) for irregular. Factory methods `Geometry.regular(dx, dy)` and `Geometry.irregular(coords)`.
- `PatternType` enum: `planar`, `rotating`, `spiral`, `source`, `sink`, `mixed`, `uncertain`
- `WaveEstimate` dataclass: `direction` (rad), `speed` (units/s), `frequency` (Hz), `wavenumber` (optional), `wavelength` (optional), `pattern_type`, `confidence` (0-1), `fit_quality` (0-1, PGD-like), `support_mask` (optional ndarray)

**`phase_gradient.py`** — Plane-wave fit + PGD (Zhang 2018)
- Read `/storage/share/codelib/sayak66--wm_travelingwaves_code` for algorithm details
- `hilbert_phase(data, axis="time")` -> unwrapped analytic phase
- `phase_gradient(phase, geometry)` -> spatial gradient via finite diff (grid) or linear regression (irregular)
- `plane_wave_fit(phase, geometry, freq=None)` -> WaveEstimate with direction, speed, fit_quality
- `pgd(phase, geometry)` -> float 0-1

**`kw_spectrum.py`** — k-w spectral estimation (no codio reference — implement from first principles)
- `kw_spectrum_3d(data, geometry, *, nperseg=None, noverlap=None)` -> xarray DataArray (kx, ky, freq)
- `kw_peaks(spectrum, n_peaks=1)` -> list[WaveEstimate]
- Uses 3D FFT with Hann windowing. Regular grids only for Phase 1.

**`optical_flow.py`** — Velocity fields (scikit-image wrappers)
- Read `/storage/share/codelib/braindynamicsusyd--neuropatttoolbox` for how they apply flow to neural data
- `compute_flow(frames, method="tvl1", **kwargs)` -> (u, v) DataArrays
- `flow_to_speed_direction(u, v)` -> speed and direction arrays

**`vector_field.py`** — Div/curl/critical points
- Read `/storage/share/codelib/braindynamicsusyd--neuropatttoolbox` for critical point classification
- `divergence(u, v, geometry)`, `curl(u, v, geometry)` -> scalar fields
- `critical_points(u, v, geometry)` -> list of (location, type) tuples
- `classify_pattern(u, v, geometry)` -> PatternType

**`beamforming.py`** — f-k for irregular arrays (no codio reference — implement from Capon 1969)
- `fk_spectrum(data, coords, freqs, slowness_grid)` -> beam power array
- `capon_beamformer(csd, coords, freqs, slowness_grid)` -> MVDR beam power
- Can use `cogpy.spectral` for cross-spectral density input

**`multitaper_nd.py`** — Separable N-D DPSS (no codio reference — implement from Hanssen 1997)
- `dpss_nd(shape, bw)` -> list of N-D taper arrays as outer products of 1-D scipy.signal.windows.dpss
- `multitaper_kw_spectrum(data, geometry, bw_time, bw_space)` -> variance-reduced k-w spectrum

**`synthetic.py`** — Test data generation
- `plane_wave(shape, geometry, direction, speed, frequency, noise_std=0)` -> DataArray (time, AP, ML)
- `spiral_wave(shape, geometry, center, angular_freq, noise_std=0)` -> DataArray
- `wave_packet(shape, geometry, direction, speed, freq, sigma_t, sigma_x, noise_std=0)` -> DataArray
- `multi_wave(components, noise_std=0)` -> superposition

**`surrogates.py`** — Null-model utilities
- Read `/storage/share/codelib/evolutionaryneuralcodinglab--travelling-waves-or-sequentially-activated-discrete-modules`
- `phase_randomize(data, axis="time")` -> preserve power spectrum, destroy phase coherence
- `spatial_shuffle(data)` -> permute spatial positions
- `surrogate_test(data, estimator_fn, n_surrogates=200, seed=None)` -> p_value, observed, null_dist

**`generalized_phase.py`** — Broadband phase stabilization
- Port from `/storage/share/codelib/mullerlab--generalized-phase`
- `generalized_phase(data, axis="time")` -> stabilized instantaneous phase for wideband signals

### Implementation order

1. `_types.py` + `synthetic.py` (needed by all tests)
2. `phase_gradient.py` + `test_phase_gradient.py`
3. `kw_spectrum.py` + `test_kw_spectrum.py`
4. `optical_flow.py` + `vector_field.py` + `test_optical_flow.py` + `test_vector_field.py`
5. `surrogates.py` + `test_surrogates.py`
6. `beamforming.py`, `multitaper_nd.py`, `generalized_phase.py` (if time permits; these are lower priority within Phase 1)
7. Update `__init__.py`

### Verification

Run `make check` at the end. All new tests must pass. Existing tests must not break.
