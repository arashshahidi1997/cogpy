docs: add Examples sections to most-used public functions

The API reference lacks inline usage examples. Almost no function has a docstring `Examples` section (only `psd_welch` does).

Action — add a NumPy-style `Examples` section to each of these functions:
1. `cogpy.preprocess.filtering.temporal.butterworth_bandpass_shoulder`
2. `cogpy.preprocess.filtering.temporal.bandpassx`
3. `cogpy.preprocess.filtering.reference.cmrx`
4. `cogpy.preprocess.filtering.normalization.zscorex`
5. `cogpy.spectral.psd.psd_multitaper`
6. `cogpy.spectral.spectrogram.spectrogramx`
7. `cogpy.detect.burst.BurstDetector` (in class or `__init__` docstring)
8. `cogpy.detect.threshold.ThresholdDetector`
9. `cogpy.measures.spatial.moran_i`
10. `cogpy.triggered.template.estimate_template`

Each example should be 3-5 lines showing a realistic call with `cogpy.datasets.load_sample()` or a small synthetic array. Keep them runnable (they'll be tested by doctest eventually). Run `make lint` after.
