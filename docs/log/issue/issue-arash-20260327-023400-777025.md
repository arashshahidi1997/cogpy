## Phase-amplitude coupling + comodulogram functions needed for pixecog coupling analysis

pixecog analysis plan §4.3 needs PAC measures (SO-spindle, SO-ripple, spindle-gamma). cogpy has no implementation; tensorpac and pactools libraries (code/lib/) have reference implementations.

### What to study

1. **tensorpac.pac.Pac** — modulation index (Tort, Ozkurt, Canolty methods), built-in surrogate testing (swap phase, time-shift, circular shift), statistical significance
2. **pactools.comodulogram.Comodulogram** — frequency x frequency PAC maps, DAR model-based PAC, multiple metrics (tort, ozkurt, canolty, penny, etc.), N_BINS_TORT=18
3. **pactools.bandpass_filter.multiple_band_pass** — efficient multi-band filtering for comodulogram computation

### What to implement in cogpy

Add to `cogpy.core.measures` or `cogpy.core.coupling`:

1. `modulation_index(phase_signal, amp_signal, n_bins=18, method='tort')` — compute PAC modulation index. Phase signal = low-freq (SO 0.5-1.25 Hz or delta), amplitude signal = high-freq (sigma, gamma, ripple). Returns MI value.

2. `preferred_phase(phase_signal, amp_signal)` — circular mean of low-freq phase at high-freq amplitude peaks. Returns preferred phase angle and mean vector length (Rayleigh test).

3. `comodulogram(signal, freq_phase_range, freq_amp_range, fs, method='tort', n_bins=18)` — frequency x frequency PAC map. Sweeps phase and amplitude frequency bands, computes MI for each pair. Returns 2D array (f_phase x f_amp).

4. `surrogate_pac(phase_signal, amp_signal, n_surrogates=200, method='circular_shift')` — generate surrogate distribution for PAC. Returns z-score of observed MI relative to surrogate distribution.

### Design constraints

- Pure functions: numpy/xarray in and out. No file paths.
- Should support both continuous signals (compute phase/amplitude internally via Hilbert) and pre-computed phase/amplitude arrays.
- Surrogate methods: circular shift (default), swap phase across trials, time-block permutation.
- Phase bins: default 18 (matching pactools convention and literature standard).

### References

- @cairney_2018_MemoryConsolidation: SO-spindle PAC, preferred phase at spindle maximum
- @stokes_2023_TransientOscillation: SO-phase histograms across full frequency range
- @peyrache_2011_InhibitionRecruitment: spindle-phase modulation of gamma

### Priority

Medium — priority 3 measure in pixecog analysis plan, needed for H3 and H6.
